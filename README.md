My Term project for my "Web Interface Design" Course.

The website consists of several pages that display information about a real estate agent and available properties. It includes:

Home Page: Featuring the agent's photo, mission statement, and customer feedback.
Property Listing Pages: Individual pages for each property with detailed information such as type, address, size, and photos.
Available Properties for Sale: A gallery-style page with thumbnail images of all the properties, including brief descriptions and links to individual property pages.
Contact Form: A form where visitors can submit inquiries, including details such as name, contact information, property type, and more.
Site Map and Credits Page: A sitemap and list of resources used for graphics and information.

Key Features:
- Clean and optimized design with a light, neutral color scheme.
- CSS styles for fonts, spacing, and layout consistency across pages.
- Intuitive navigation through a common look-and-feel approach.
- Easy-to-read text and responsive layout for a better user experience on various devices.

This project demonstrates my ability to create a multi-page, visually appealing, and functional website.
